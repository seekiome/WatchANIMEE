# WatchTogether 🎬

Синхронный просмотр видео с друзьями. Только хост загружает файл — друзья смотрят по ссылке.

## Как задеплоить на Railway (бесплатно)

1. Залей этот репозиторий на GitHub
2. Зайди на [railway.app](https://railway.app) → войди через GitHub
3. Нажми **New Project → Deploy from GitHub repo**
4. Выбери этот репозиторий
5. Railway автоматически запустит `npm start`
6. Зайди в **Settings → Networking → Generate Domain** — получишь ссылку

## Использование

1. Открой сайт → нажми **Хост**
2. Нажми 🔗 чтобы скопировать ссылку и отправить другу
3. Загрузи видео — друг увидит его автоматически
4. Пауза/плей синхронизируются у всех

## Технологии

- Node.js + Express
- WebSocket (ws) для синхронизации
- Multer для загрузки файлов
- Чистый HTML/CSS/JS на фронте
